How Scypher Works

A revolutionary approach to seed phrase security using XOR encryption and blockchain storage

1

XOR Encryption

Your BIP39 seed phrase is encrypted using XOR cipher with SHAKE-256 key derivation

2

Blockchain Storage

The cipher code is split, encoded, and stored as token descriptions on Ergo blockchain

3

Decentralized Access

Retrieve the complete script anytime from the blockchain - no servers needed

4

Local Execution

Run the script locally on your machine for maximum security and privacy
